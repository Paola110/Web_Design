Creacion de tu primer pagina web

Que pasos debes seguir para crear tu primera pagina web?

Para crear tu primera pagina web debes seguir los siguientes pasos:
1. Abrir un editor de texto como por ejemplo notepad++ o sublime text.
2. Escribir el codigo HTML basico.
3. Guardar el archivo con la extension .html.
4. Abrir el archivo en un navegador web.
